    # 表单验证

    微信小程序表单验证需要在onReady中绑定

    onReady() {
    // 需要在onReady中设置规则
    this.$refs.form.setRules(this.rules)
},

    需要注意的是要在绑定的表单中写明数据项的名字!! 否则不起作用或验证错误,重置表单的时候也千万要注意
